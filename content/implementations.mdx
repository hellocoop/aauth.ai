# Implementations

## TypeScript

The reference implementation of AAuth in TypeScript:

- [hellocoop/AAuth](https://github.com/hellocoop/AAuth) — TypeScript implementation of the AAuth protocol family.

## Other Implementations

| Language | Repository | Description |
|----------|------------|-------------|
| **Python** | [christian-posta/aauth-full-demo](https://github.com/christian-posta/aauth-full-demo) | Full demo with agent, resource, and AS |
| **Java (Keycloak)** | [christian-posta/keycloak-aauth-extension](https://github.com/christian-posta/keycloak-aauth-extension) | Keycloak extension for AAuth |

Christian Posta's [AAuth Full Demo](https://blog.christianposta.com/aauth-full-demo/) is a working end-to-end environment with implementations in Java, Python, and Rust — using Keycloak as the IdP and [Agentgateway](https://github.com/agentgateway/agentgateway) for policy enforcement.

*Building an AAuth implementation? [Open an issue](https://github.com/hellocoop/aauth.dev/issues) to get listed here.*
